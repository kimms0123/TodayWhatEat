# 오늘 뭐 먹지 (프로토타입)

상황/재료를 입력하면 **Gemini 기반으로 메뉴/레시피를 추천**하고, 배달 모드는 **배민 딥링크 검색**까지 연결하는 MVP 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열면 됩니다.

## 환경변수 (선택)

Gemini를 실제로 호출하려면 `.env.local`에 아래 값을 설정하세요.

```bash
GEMINI_API_KEY=YOUR_KEY
```

키가 없으면 **목업 응답**으로도 화면이 동작합니다.

## 화면

- `/`: 홈(모드 선택)
- `/delivery`: 배달 모드(상황/필터 → 추천 3개 → 배민 딥링크)
- `/fridge`: 냉털 모드(재료 입력 → 10분 레시피 1~2개)
